### Backend 
```
$ git clone https://github.com/2024-Summer-Bootcamp-TeamJ/Backend.git
```
### env setting in the Backend folder
* backend/.env
```
# mysql
DATABASE_URL=
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=

# CELERY
CELERY_BROKER_URL=

# openai
OPENAI_API_KEY=
GPT_MODEL=

# AWS OpenSearch
OPENSEARCH_URL=
OPENSEARCH_ADMIN=
OPENSEARCH_PASSWORD=

# Administer
ADMIN_USERNAME=
ADMIN_PASSWORD=

# Logging
LOG_LEVEL=
```
### Run Docker
```
docker-compose up -d 
```
### Frontend
```
git clone https://github.com/2024-Summer-Bootcamp-TeamJ/Frontend.git
```
### Install
```
yarn install
```

### Run
```
yarn dev
```
